Frisbee Flight Simulator

Instructions:

  • Speed field should only allow numbers from 2 to 14
  • Glide field should only allow numbers from 2 to 8
  • Turn field should only allow numbers from -5 to 1
  • Fade field should only allow numbers from 0 to 5

When the form is submitted take the values in the form and calculate the distance in feet of the frisbee's flight using this formula:

distance = 10 * (speed * glide) + (turn * -1.2) + (fade * 0.5)

Display the result in the div with the id of result.

If the results don't make a ton of sense thats okay, this formula is not accurate.

If the form is submitted and there are any invalid values display a list of errors in the div with the id of errors.
